About This Project

Methodology, data sources, and limitations.

Data Source

This project uses the Income-Restricted Housing Inventory (2022) published by the City of Boston's Mayor's Office of Housing (MOH) on Analyze Boston.

The dataset includes {{ total }} projects covering public housing owned by the Boston Housing Authority (BHA), privately-owned housing built with city funding, and privately-owned housing created through programs like Low-Income Housing Tax Credits (LIHTC) or the Inclusionary Development Policy (IDP).

The database only includes units that have a deed-restriction dictating their maximum rents or prices.

Geocoding

The original dataset provides only project name, neighborhood, and zip code — no street addresses or coordinates. To place projects on the map, we geocoded all {{ total }} projects using the Google Geocoding API.

{{ geocoded_count }} of {{ total }} projects ({{ "%.1f"|format(geocoded_count / total * 100) }}%) were successfully mapped to addresses. {{ total - geocoded_count }} projects could not be reliably geocoded and appear in the data table only.

Many project names in the dataset use anonymized street numbers (e.g., "xxxx Blue Hill Ave"). The geocoder resolves these to the correct street but not the exact building number.

Limitations

  • Data reflects the 2022 inventory and may not include recent additions or closures.
  • Geocoded locations are approximate for projects without specific addresses.
  • Some project names are anonymized ("Suppressed") in the source data for privacy reasons.
  • The dataset does not include market-rate housing or housing that lacks a legal deed restriction.

Technology

  • Backend: Python / Flask
  • Map: Leaflet.js with MarkerCluster
  • Search: Fuse.js fuzzy matching
  • Geocoding: Google Geocoding API
  • Data: City of Boston Open Data (Analyze Boston)
